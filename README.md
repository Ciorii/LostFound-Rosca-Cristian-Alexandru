## Project Title / Titlu proiect
Lost&Found App

## Description / Descriere
Aplicatie Android pentru raportarea si cautarea obiectelor pierdute sau gasite. Utilizatorii pot adauga postari cu locatie, poze si contact, iar lista poate fi cautata dupa titlu. /  
Android app for reporting and searching lost or found objects. Users can add posts with location, photos and contact info, and the list can be searched by title.

## Features / Functionalitati
- autentificare user (login/register) / user authentication (login/register)  
- lista de postari (lost/found) / posts list (lost/found)  
- adaugare postare cu locatie pe harta / add post with map location  
- upload poze si afisare imagini / upload and display images  
- cautare dupa titlu / search by title  
- detalii postare + contact + harta / post details + contact + map  
- stergere postare de catre autor / delete post by owner  

## Technologies Used / Tehnologii folosite
- Java 
- Android SDK
- Firebase Authentication 
- Firestore (via server)  
- REST API (ASP.NET Core) 
- Retrofit
- Glide (image loading)  
- Google Maps SDK 
- ImgBB (image upload)
